5 Bias Dimensions — Cialdini's Principles of Influence + MITRE ENGAGE Adversary Engagement
Curiosity Gap
Incomplete secrets that compel completion. Truncated credentials, partial keys, and breadcrumb paths that lead deeper into the honeypot.
Signals: find *.key · cat .env · ls /root
Confirmation Bias
Fake evidence matching attacker beliefs. If they expect a financial target, plant convincing financial data that confirms it.
Signals: uname -a · cat /etc/os-release · whoami
Sunk Cost Trap
Reward early effort with small wins so attackers won't leave empty-handed — making them invest more time and reveal more infrastructure.
Signals: wget · git clone · pip install
Authority Signals
Fabricated C-level memos, emergency admin credentials, and audit documents that demand immediate action and override caution.
Signals: sudo · cat /etc/shadow · authorized_keys
Scarcity Framing
Urgency cues — "DELETE BEFORE AUDIT" files, countdown timestamps — trigger rushed, careless, identity-revealing behavior.
Signals: crontab · nohup · screen · disown
