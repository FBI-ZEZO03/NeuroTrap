FedAvg Protocol
1
Local Training
Each node trains on its own honeypot sessions. Raw data stays local — only gradient deltas computed.
2
DP Noise Injection
Gaussian noise (ε=1.0) added to gradients before transmission. Individual data becomes mathematically unrecoverable.
3
FedAvg Aggregation
Server averages all deltas using weighted FedAvg. No single organization's data is recoverable from the aggregate.
4
Global Model Update
Improved weights distributed to all nodes. Every organization gets smarter from collective intelligence.

Academic Basis & Implementation Stack
IEEE S&P Reference
McMahan et al., "Communication-Efficient Learning of Deep Networks from Decentralized Data" — the original FedAvg paper that FHIM implements.
Differential Privacy
Gaussian mechanism (ε=1.0, δ=1e-5) via Opacus. Provides (ε,δ)-DP guarantees: any single node's data contribution is indistinguishable from noise.
