5 Bias Dimensions — Cialdini's Principles + MITRE ENGAGE
Curiosity Gap
Incomplete secrets that compel completion. Truncated credentials and breadcrumb paths lead attackers deeper.
Triggers: find *.key · cat .env · ls /root
Confirmation Bias
Fake evidence matching attacker beliefs. Plant data confirming their expected target environment.
Triggers: uname -a · cat /etc/os-release
Sunk Cost Trap
Reward early effort so attackers won't leave empty-handed — making them invest more time and reveal more.
Triggers: wget · git clone · pip install
Authority Signals
Fabricated C-level memos and emergency credentials that override attacker caution.
Triggers: sudo · cat /etc/shadow
Scarcity Framing
Urgency cues trigger rushed, careless, identity-revealing behavior from attackers.
Triggers: crontab · nohup · screen
